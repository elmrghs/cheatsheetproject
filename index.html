fundamentals of html

HTML, hypertext markup language: markup language defines the structure and presentation of raw text → computers interpret raw text wrapped in elements. hypertext is text displayed on a device that acts as a gateway to other text through links, aka hyperlinks..

semantic tags make websites more accessible for those who use screen readers and improves site SEO. they are not visible elements on the webpage, more like a categorisation system.

tag definition
<header> is a container most commonly used for navigational links or introductory content (h1-h6 often used here)
<nav> defines a block of navigation links (menus, table of contents…); can be used inside of the < header > element and on it’s own
<main> used to encapsulate the dominant content within a webpage. using < main > enables screen readers/browsers to better identify the priority/bulk content.
<footer> indicates the content at the bottom of the webpage; normally contains contact info, copyright info, terms of use, site map, reference to top of page links

below is a list of common tags used in html, that will eventually become second nature.

tag used for
<h1> the heading tag always starts with 'h' and runs from h1 to h6, 1 being the largest heading and 6 being the smallest.
<p> the paragraph tag is used for body text, things that are not titles to something else.
<a> the anchor tag is used to link to internal pages, external pages or content on the same page.
<ol> the ordered list tag creates a numbered list.
<ul> the unordered list tag creates a list marked with bullet points.
<l> the list tag is used within an ordered (ol) or unordered list (ul).
<div> the division tag is a hidden tag used for creating enclosed blocks of html.
<span> the span tag is a hidden tag used to separate small pieces of content that share the same line as other content.

fundamentals of css

CSS, cascading style sheets: styling language used to style html content, such as colours, font types, don't sizes, images, positioning..etc.

selectors identify the specific html element to be styled by the declaration.

selector used for
type, p the type selector matches a given type of the element in the html document; does not need the < >; can also be tag name or element selector
universal, * the universal selector selects all elements of any type; uses * character in place of type selector
class, .class the class selector is used to select an element in html/css with a class attribute; to select an html element with a class, it should be .classname
id, #id id tags should be used when an element needs to be styled uniquely; can only have a single value; used once per page; found using #
attribute, [attr] attribute selector used to target html element that already contain attributes
valued attribute, [attr*='value'] used to select an element where the attribute contains any instance of the specified value
pseudo-class, :hover pseudo-class selectors are used to style a changing element, for example: when the submit button is greyed out until all the required fields are filled out

properties are the first part of a css declaration. they are used to signify what visual characteristic of the element will be modified.

property used for
font-family tells browser what typeface to display
font-size tells the browser what size font to use
text-align tells the browser where to align the text. start, end, left, right, center or justify can be used
color changes the foreground color
background-color changes the background color
opacity used to dictate the opacity of an element, uses integers
text-decoration applies a decoration or clears it from a piece of text. none, underline, overline, line-through or blink can be used
display helps to position elements on the page. google this as there are many options
padding applies padding around an element. top, left, bottom, middle can be used
margin applies a margin to an element. top, left, bottom, middle can be used
border applies a border to an element. top, left, bottom, middle can be used
